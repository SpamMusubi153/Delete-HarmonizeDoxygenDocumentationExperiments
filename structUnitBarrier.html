Detailed Description

template<typename OP_SET, typename ADR_TYPE>
struct UnitBarrier< OP_SET, ADR_TYPE >

The UnitBarrier template struct is a barrier that can hold up to one promise with an operation type contained by its operation set.

The UnitBarrier template struct acts as a barrier for a single await. This is useful for setting up additional layers of resolution for multi-dependency awaits.
