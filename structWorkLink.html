Detailed Description

template<typename OP_SET, typename ADR_TYPE, size_t GROUP_SIZE>
struct WorkLink< OP_SET, ADR_TYPE, GROUP_SIZE >

The WorkLink template struct, given a PromiseUnion union, an address type, and a group size, stores an array of GROUP_SIZE promise unions of the corresponding type and an address value of type ADR_TYPE. Instances of this template also contain a Op value to identify what type of work is contained within the link, a meta_data field, and a count field to indicate the number of contained promises.
