Detailed Description

template<typename HEAD, typename... TAIL>
struct PromiseUnion< OpUnion< HEAD, TAIL... > >::Info

For some compilers, a union cannot diretly contain static const values. To get around this, we wrap our compile time type information for PromiseUnions with internl types such as the Info struct type or the Looup template struct.
