Detailed Description

template<typename PROGRAM, typename TYPE>
struct Return< PROGRAM, TYPE >

The Return template struct is used to represent the set of all types that can be returned from an operation's eval function. This includes returning by value, returning with a promise that returns the appropriate type, or returning a future that will eventually define a value of the appropriate type.

The return_guard function causes a compile time error when promises that do not belong in the operation set of a Return are passed in as initializing arguments. This helps to replace the more cryptic errors that come about when trying to pass in an inappropriate Promise type.
