Detailed Description

template<typename OPERATION>
struct Promise< OPERATION >

The Promise template struct represents all information associated with an async call to its opertion type, including all passed arguments as well as return addresses.

The Promise template struct represents the arguments and return information associated with an asynchronous call to the input operation type parameter.
