Detailed Description

template<typename HEAD, typename... TAIL>
union PromiseUnion< OpUnion< HEAD, TAIL... > >

The recursive case of the PromiseUnion template union defines the cast(), rigid_eval(), and loose_eval() template functions for the async function/parameter type corresponding to the first template argument.
